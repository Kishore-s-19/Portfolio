# Portfolio
A collection of my full-stack and frontend projects showcasing modern web development, interactive UI/UX, animations, and scalable architectures. Built using React, Vite, Spring Boot, MySQL, and motion-driven design to solve real-world problems.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, GSAP
- **Backend**: Java, Spring Boot, MySQL
- **Tools**: Git, GitHub, VS Code

## Features
- **Dynamic UI/UX**: High-end animations and interactive components.
- **Responsive Design**: Optimized for all devices.
- **Modern Assets**: Custom-designed elements and smooth transitions.
